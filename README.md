#  Smart Task Manager

## ✨ Features

### 🎯 Core Functionality
- ✅ Add, edit, and delete tasks
- 🔄 Mark tasks as complete/incomplete
- 📅 Set due dates with overdue alerts
- 🏷️ Categorize tasks
- 🎨 Priority levels (High, Medium, Low) with color coding

### 🔍 Advanced Features
- 🔎 Real-time search through tasks and categories
- 🏷️ Filter by priority and completion status
- 📅 Sort tasks by due date
- 💾 Local storage persistence (no data loss)
- 📱 Responsive design for all devices

### 🎨 Modern UI/UX
- 🌟 3D glassmorphism design
- ✨ Smooth animations and hover effects
- 🎭 Gradient backgrounds and glowing elements
- 📐 Neumorphism-style components
- 🌈 Color-coded priorities with visual effects

## 🛠️ Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Modern styling with 3D effects, animations, and glassmorphism
- **JavaScript (ES6+)** - Interactive functionality and DOM manipulation
- **Local Storage API** - Client-side data persistence


## 📖 Usage

### Adding Tasks
1. Enter task description in the input field
2. Select priority level (High/Medium/Low)
3. Set due date (optional)
4. Add category (optional)
5. Click "Add Task" or press Enter

### Managing Tasks
- **Complete**: Click the checkbox next to a task
- **Edit**: Double-click on task text to edit
- **Delete**: Click the "Delete" button
- **Search**: Use the search bar to find tasks
- **Filter**: Use dropdowns to filter by priority or status
- **Sort**: Click "Sort by Due Date" to organize tasks

### Visual Indicators
- 🔴 **High Priority**: Red with glow effect
- 🟠 **Medium Priority**: Orange with glow effect
- 🟢 **Low Priority**: Green with glow effect
- ⚠️ **Overdue**: Pulsing red animation
- ✅ **Completed**: Strikethrough with reduced opacity
